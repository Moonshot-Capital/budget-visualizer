# Budget Visualizer 📊

Ein interaktives, responsives und modernes Budget-Visualisierungs-Tool zur Verwaltung deiner monatlichen Einnahmen und Ausgaben. 

Dieses Tool wurde mit Fokus auf Ästhetik, Benutzerfreundlichkeit und flüssige Animationen entwickelt.

## ✨ Features
- **Interaktive Visualisierung**: Ein dynamischer Balken zeigt deine Ausgaben im Verhältnis zu deinem Einkommen.
- **Drag-and-Drop**: Sortiere deine Ausgaben ganz einfach per Ziehen und Ablegen, um die Reihenfolge im Balken anzupassen.
- **Inline-Bearbeitung**: Doppelklicke auf eine Ausgabe, um ihren Namen oder Betrag direkt anzupassen.
- **Schnelles Hinzufügen (Quick Add)**: Füge häufige Ausgaben mit nur einem Klick hinzu.
- **Dynamischer Überschuss/Defizit**: Sieht auf einen Blick, ob du im Plus oder Minus bist.
- **Responsive Design**: Optimiert für Desktop-, Tablet- und Smartphone-Bildschirme (keine überlappenden Labels auf Mobilgeräten).
- **Offline-First**: Alle Daten werden sicher in deinem lokalen Browser-Speicher (`localStorage`) gespeichert.


## 🛠️ Technologien
- **HTML5** (Semantische Struktur)
- **Vanilla CSS3** (Styling mit Variablen, CSS-Grid, Flexbox und Animationen)
- **Vanilla JavaScript** (Zustandsverwaltung, Drag-and-Drop API, Positionsberechnungen)
